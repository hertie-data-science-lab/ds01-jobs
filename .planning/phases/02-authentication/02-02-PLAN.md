Brownfield reference: src/auth.py contains a working HMAC auth flow. Key differences from brownfield:
1. Key format: `ds01_` + 32 bytes base64url (NOT `ds01_<username>_<random32>`)
2. Key lookup: by key_id column (first 8 chars of base64url portion), then bcrypt verify
3. All auth failures: generic "Authentication failed" (locked decision) - log specific reason server-side
4. Expiry warning: ISO date `X-DS01-Key-Expiry-Warning: 2026-06-04` (not relative days)
5. bcrypt: use `bcrypt` directly (not passlib CryptContext)
6. bcrypt in async: use `asyncio.to_thread(bcrypt.checkpw, ...)` to avoid blocking event loop

  <name>Task 1: Create HMAC auth dependency with nonce replay protection</name>
  <files>src/ds01_jobs/auth.py, tests/unit/test_auth.py</files>
  <action>
  Create src/ds01_jobs/auth.py implementing the full HMAC-SHA256 auth flow:

  **Module-level constants:**
  - `HMAC_TOLERANCE_SECONDS = 300`
  - `NONCE_EXPIRY_SECONDS = 300`
  - `KEY_EXPIRY_WARNING_DAYS = 14`
  - `security = HTTPBearer()`

  **Nonce cache (module-level):**
  - `_used_nonces: dict[str, float] = {}` mapping nonce to expiry timestamp (time.monotonic)
  - `_cleanup_nonces()` - remove expired entries (called on every auth request)
  - `_check_and_store_nonce(nonce: str) -> bool` - returns True if fresh, False if replay

  **Key lookup:**
  - `async def _get_key_record(db: aiosqlite.Connection, key_id: str) -> aiosqlite.Row | None`
  - Query: `SELECT * FROM api_keys WHERE key_id = ? AND revoked = 0` (revoked check at query level)
  - Returns row with key_hash, username, expires_at, etc.

  **HMAC helpers:**
  - `_build_canonical(method, path, timestamp, nonce, body) -> str` - format: METHOD\nPATH\nTIMESTAMP\nNONCE\nBODY_SHA256_HEX
  - `_verify_signature(raw_key, canonical, provided_signature) -> bool` - HMAC-SHA256 with hmac.compare_digest

  **Main dependency `get_current_user`:**
  ```python
  async def get_current_user(
      request: Request,
      response: Response,
      credentials: HTTPAuthorizationCredentials = Depends(security),
      db: aiosqlite.Connection = Depends(get_db),
  ) -> dict[str, str]:
  ```

  Flow:
  1. Extract raw_key from credentials.credentials
  2. Validate prefix: must start with `ds01_`. If not, raise 401.
  3. Extract key_id: characters 5 through 13 of raw_key (first 8 chars of base64url portion)
  4. Look up key record by key_id. If not found, raise 401.
  5. Check expiry (ISO 8601 UTC comparison). If expired, log reason + username, raise 401.
  6. bcrypt verify: `await asyncio.to_thread(bcrypt.checkpw, raw_key.encode(), row["key_hash"].encode())`. If fails, log reason, raise 401.
  7. Extract X-Timestamp, X-Nonce, X-Signature headers. If any missing, raise 401.
  8. Timestamp freshness check (within 300s of now). If stale, log reason + username, raise 401.
  9. Nonce replay check via `_check_and_store_nonce`. If replay, log reason + username, raise 401.
  10. Build canonical string, verify HMAC signature. If invalid, log reason + username, raise 401.
  11. Update last_used_at: `await db.execute("UPDATE api_keys SET last_used_at = ? WHERE key_id = ?", (now_iso, key_id))` then `await db.commit()`
  12. Set expiry warning header if within 14 days: `response.headers["X-DS01-Key-Expiry-Warning"] = expires_at.date().isoformat()` (ISO date only, e.g. "2026-06-04")
  13. Return `{"username": row["username"]}`

  **All 401 responses:** Use generic `HTTPException(status_code=401, detail="Authentication failed")`. Log the specific reason via `logging.warning(f"Auth failed for {username}: {reason}", extra={"ip": request.client.host})` (use stdlib logging).

  Create tests/unit/test_auth.py:
  - Test valid HMAC signature passes (create a key in test DB, sign a request, verify auth succeeds)
  - Test invalid key prefix returns 401
  - Test expired key returns 401
  - Test revoked key returns 401 (revoked=1 in DB)
  - Test stale timestamp returns 401
  - Test nonce replay returns 401
  - Test invalid HMAC signature returns 401
  - Test expiry warning header set when key expires within 14 days (ISO date format)
  - Test expiry warning header NOT set when key has >14 days remaining
  - Test last_used_at updated after successful auth

  Use pytest-asyncio, httpx.AsyncClient with app.dependency_overrides or TestClient. Create helper functions to generate valid HMAC signatures for test requests.
  </action>
